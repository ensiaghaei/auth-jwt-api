# Auth JWT API

Spring Boot JWT Authentication API with Docker and PostgreSQL.

## Features
- User Registration
- User Login
- JWT Authentication
- Role-based Authorization
- Docker Support
- PostgreSQL Database

## Technologies
- Java 21
- Spring Boot
- Spring Security
- JWT
- PostgreSQL
- Docker
- Maven

## Run with Docker

```bash
docker compose up --build
```
## API Endpoints
### Register
POST /api/auth/register
### Login
POST /api/auth/login
